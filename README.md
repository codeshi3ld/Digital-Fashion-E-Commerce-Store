## Overview

A3RO is a modern and responsive e-commerce web application focused on footwear. The platform allows users to browse, explore, and shop for shoes across different categories including Men's, Women's, and Kids. It is designed with a clean UI, intuitive navigation, and a user-friendly shopping experience.

This project demonstrates core front-end development skills, including layout structuring, product display, and category-based navigation.

---

## Features

* Clean and responsive user interface
* Category-based navigation (Footwear, Men's, Women's, Kids)
* Product listing with images, price, and variations
* Highlight sections (e.g., Superstar, Top Seller, New Arrivals)
* Search functionality
* Basic cart and notification UI (icons)
* Footer with customer support and store information
